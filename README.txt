
Rasha Adel - Final Static Management UI
--------------------------------------
This is a fully static, offline-ready frontend (HTML/CSS/JS) for Rasha Adel Beauty Center & Spa.

How to use:
1. Unzip the package.
2. Open index.html in your browser or open the folder in VS Code and use Live Server.
3. The app stores data in localStorage (users, services, clients, invoices).
4. Default admin account:
   - username: rashaadel23
   - password: rashaadel2333
5. Admin can add staff accounts via "Manage Users".
6. Staff can register customers and create invoices.
7. Replace assets/logo11.jpg and assets/background.jpg with your final images.
